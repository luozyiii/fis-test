# fis-test

> 安装fis3

    npm install -g fis3

> 升级fis3
    
    npm install -g fis3

> 命令

    fis3 release -d <path>
    <path> 任意目录
    fis3 release -h 获取更多参数

    eg:
    构建发布到项目目录的 output 目录下
    fis3 release -d ./output

    构建发布到项目父级目录的 dist 子目录下
    fis3 release -d ../dist

    发布到其他盘 （Windows）
    fis3 release -d D:\output
